**Step 1: Fix DatabaseManager to support multi-statement schema files.**

Current `runMigrations()` uses `stmt.executeUpdate(schema)` which only runs one SQL statement. Replace the migration execution with semicolon-split logic:

```java
private void runMigrations() {
    String resourcePath = isMysql ? "/schema/schema_mysql.sql" : "/schema/schema_sqlite.sql";
    String schema = readResource(resourcePath);
    try (Connection conn = dataSource.getConnection();
         Statement stmt = conn.createStatement()) {
        for (String sql : schema.split(";")) {
            String trimmed = sql.trim();
            if (!trimmed.isEmpty()) {
                stmt.executeUpdate(trimmed);
            }
        }
        JBalance.LOGGER.info("[JBalance] Schema migration completed successfully");
    } catch (SQLException e) {
        throw new RuntimeException("[JBalance] Schema migration failed", e);
    }
}
```

```java
package com.pweg0.jbalance.data.db;

import javax.sql.DataSource;
import java.sql.*;
import java.util.*;

public class PlaytimeRepository {

    public record PlaytimeData(long activeSeconds, Set<Long> claimedHours) {}

    private final DataSource dataSource;
    private final boolean isMysql;

    public PlaytimeRepository(DataSource dataSource, boolean isMysql) {
        this.dataSource = dataSource;
        this.isMysql = isMysql;
    }

    /** Loads playtime data for a player. Returns null if no record exists. */
    public PlaytimeData loadPlaytime(UUID uuid) {
        String sql = "SELECT active_seconds, claimed_hours FROM jbalance_playtime WHERE uuid = ?";
        try (Connection c = dataSource.getConnection();
             PreparedStatement ps = c.prepareStatement(sql)) {
            ps.setString(1, uuid.toString());
            try (ResultSet rs = ps.executeQuery()) {
                if (rs.next()) {
                    long seconds = rs.getLong("active_seconds");
                    Set<Long> claimed = parseClaimedHours(rs.getString("claimed_hours"));
                    return new PlaytimeData(seconds, claimed);
                }
                return null;
            }
        } catch (SQLException e) {
            throw new RuntimeException("loadPlaytime failed for " + uuid, e);
        }
    }

    /** Upserts playtime data. MySQL uses INSERT ... ON DUPLICATE KEY UPDATE; SQLite uses INSERT OR REPLACE. */
    public void upsertPlaytime(UUID uuid, long activeSeconds, String claimedHoursStr) {
        String sql = isMysql
            ? "INSERT INTO jbalance_playtime (uuid, active_seconds, claimed_hours) VALUES (?, ?, ?) ON DUPLICATE KEY UPDATE active_seconds = VALUES(active_seconds), claimed_hours = VALUES(claimed_hours)"
            : "INSERT OR REPLACE INTO jbalance_playtime (uuid, active_seconds, claimed_hours) VALUES (?, ?, ?)";
        try (Connection c = dataSource.getConnection();
             PreparedStatement ps = c.prepareStatement(sql)) {
            ps.setString(1, uuid.toString());
            ps.setLong(2, activeSeconds);
            ps.setString(3, claimedHoursStr);
            ps.executeUpdate();
        } catch (SQLException e) {
            throw new RuntimeException("upsertPlaytime failed for " + uuid, e);
        }
    }

    private static Set<Long> parseClaimedHours(String str) {
        Set<Long> set = new HashSet<>();
        if (str == null || str.isEmpty()) return set;
        for (String part : str.split(",")) {
            String trimmed = part.trim();
            if (!trimmed.isEmpty()) {
                try { set.add(Long.parseLong(trimmed)); } catch (NumberFormatException ignored) {}
            }
        }
        return set;
    }
}
```

```java
package com.pweg0.jbalance.service;

import com.pweg0.jbalance.JBalance;
import com.pweg0.jbalance.config.JBalanceConfig;
import com.pweg0.jbalance.data.db.PlaytimeRepository;
import com.pweg0.jbalance.util.CurrencyFormatter;
import net.minecraft.network.chat.Component;
import net.minecraft.server.level.ServerPlayer;

import java.util.*;
import java.util.concurrent.*;

public class PlaytimeService {

    private static PlaytimeService instance;

    private final PlaytimeRepository repo;
    private final ExecutorService dbExecutor;

    // Per-player in-memory state
    private final ConcurrentHashMap<UUID, PlayerState> playerStates = new ConcurrentHashMap<>();

    // Periodic flush counter (ticks)
    private long flushTickCounter = 0;
    private static final long FLUSH_INTERVAL_TICKS = 6000; // 5 minutes = 5 * 60 * 20

    private record PlayerState(
        long activeSeconds,
        Set<Long> claimedHours,
        double lastX, double lastY, double lastZ,
        float lastYRot, float lastXRot,
        long ticksSinceLastMove
    ) {}

    public PlaytimeService(PlaytimeRepository repo, ExecutorService dbExecutor) {
        this.repo = repo;
        this.dbExecutor = dbExecutor;
        instance = this;
    }

    public static PlaytimeService getInstance() {
        return instance;
    }
```

**Core methods:**

**`onLogin(ServerPlayer player)`** — called from EarningsEventHandler.onPlayerLoggedIn:
- Load playtime from DB async: `CompletableFuture.supplyAsync(() -> repo.loadPlaytime(uuid), dbExecutor)`
- On completion (back on game thread via `server.execute()`): store in `playerStates` map
- If no DB record, create fresh state with 0 seconds and empty claimed set
- Initialize position tracking from player's current position

**`onLogout(UUID uuid)`** — called from EarningsEventHandler.onPlayerLoggedOut:
- Get state from `playerStates`, remove it
- Flush to DB async: `CompletableFuture.runAsync(() -> repo.upsertPlaytime(uuid, state.activeSeconds, formatClaimedHours(state.claimedHours)), dbExecutor)`

**`onTick(ServerPlayer player)`** — called every server tick per player:
- Get state from `playerStates`; if null, return (still loading from DB)
- Compare current position/rotation to stored position:
  - `player.getX(), player.getY(), player.getZ(), player.getYRot(), player.getXRot()`
  - If ANY value changed: reset `ticksSinceLastMove` to 0, update stored position
  - If ALL values same: increment `ticksSinceLastMove`
- Read AFK timeout on game thread: `long afkTimeoutTicks = JBalanceConfig.AFK_TIMEOUT_MINUTES.get() * 60 * 20;`
- If `ticksSinceLastMove >= afkTimeoutTicks`: player is AFK, do NOT increment active time, return
- Otherwise: every 20 ticks (1 second boundary), increment `activeSeconds` by 1
  - Use `(totalTicksTracked % 20 == 0)` — track a separate tick counter or check if activeSeconds should advance based on ticks accumulated
  - Simpler approach: track `activeTicks` instead of `activeSeconds`, increment every non-AFK tick, derive seconds as `activeTicks / 20`
- After incrementing, check milestones:
  - Read milestones on game thread: `List<JBalanceConfig.MilestoneEntry> milestones = JBalanceConfig.parsedMilestones();`
  - For each milestone: if `activeSeconds >= milestone.hours() * 3600` AND `!claimedHours.contains(milestone.hours())`:
    - Add to claimedHours set (in-memory, authoritative during session)
    - Credit coins: `EconomyService.getInstance().give(uuid, milestone.reward())`
    - Send milestone notification (on game thread, we're already on it):
      ```java
      player.sendSystemMessage(Component.literal(
          "§6[JBalance] §aVoce completou " + milestone.hours() + "h de jogo! Recompensa: §6" + CurrencyFormatter.formatBalance(milestone.reward())
      ));
      ```
- IMPORTANT: Update playerStates map with new state (since record is immutable, create new record)
  - Alternative: use a mutable class instead of record for PlayerState to avoid excessive object creation every tick

**DESIGN DECISION:** Use a mutable inner class instead of record for PlayerState to avoid creating a new object every tick:
```java
private static class PlayerState {
    long activeTicks;       // non-AFK ticks since tracking started
    final Set<Long> claimedHours;
    double lastX, lastY, lastZ;
    float lastYRot, lastXRot;
    long ticksSinceLastMove;

    PlayerState(long activeTicks, Set<Long> claimedHours,
                double x, double y, double z, float yRot, float xRot) {
        this.activeTicks = activeTicks;
        this.claimedHours = claimedHours;
        this.lastX = x; this.lastY = y; this.lastZ = z;
        this.lastYRot = yRot; this.lastXRot = xRot;
        this.ticksSinceLastMove = 0;
    }

    long activeSeconds() { return activeTicks / 20; }
}
```

**`onServerTick()`** — periodic flush, called from EarningsEventHandler.onServerTick:
- Increment flushTickCounter
- If `flushTickCounter >= FLUSH_INTERVAL_TICKS` (6000 ticks = 5 minutes):
  - Reset counter
  - For each entry in playerStates: flush to DB async
  - Log: `JBalance.LOGGER.debug("[JBalance] Periodic playtime flush for {} players", playerStates.size())`

**`flushAll()`** — called on server stopping:
- For each entry in playerStates: flush to DB synchronously (use `CompletableFuture.runAsync(...).join()` or direct repo call since server is shutting down anyway)
- Clear playerStates

**`formatClaimedHours(Set<Long> hours)`** — converts set to comma-delimited string: `hours.stream().sorted().map(String::valueOf).collect(Collectors.joining(","))`
